# Kinova Gen3 7-DoF Motion Planner & Animator

This repository provides forward/inverse kinematics, joint-space trajectory generation, and 3D animation for the Kinova Gen3 Ultra lightweight 7-DoF spherical wrist robot.

## Quick start

```bash
git clone https://github.com/you/gen3_motion.git
cd gen3_motion
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python tests.py
